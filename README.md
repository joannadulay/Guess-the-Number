# Guess-the-Number
It is a simple game about guessing the number between 1 and 100 with only seven attempts built as a desktop application using .NET. It will give you a clue whether the number you guessed was too high or too low. Have fun!

## Features

- Graphical user interface (GUI)
- Random number generation
- Input validation
- Feedback for each guess (e.g., too high or too low)
- Simple and beginner-friendly gameplay

## Built With

- C#
- .NET 10.0
- Windows Desktop

## Requirements

- .NET 10 Runtime (or compatible version)
- Windows operating system

## Getting Started

1. Download or clone the repository.
2. Ensure the .NET 10 Desktop Runtime is installed.
3. Run:

```
NumberGuessingGameGUI.exe
```

## How to Play

1. Launch the application.
2. Enter a number into the input field.
3. Submit your guess.
4. Read the feedback provided by the application.
5. Continue guessing until you find the correct number.
